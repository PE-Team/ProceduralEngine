ADDING A NEW SUBMODULE/DEPENDENCY
>git submodule add -f repository_path local_file_path

Note: do this in the main directory of the repository/project

AFTER EACH CHECKOUT AND CREATION DO:
>git submodule init
>git submodule update

Note: do this in the main directory of the repository/project

WHEN YOU WANT TO UPDATE DO:
>cd local_file_path
>git pull

Note: you change directory into the submodule/dependency's directory and then pull
